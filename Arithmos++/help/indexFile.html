File Menu

Back:  Toolbar       Next:  Edit Menu       Up:  File Window Contents

New file (Ctrl-N) : creates a new source file.

Open file (Ctrl-O) : loads a source file from disk.

Save file Window (Ctrl-S) : saves the current source file to disk.

Save file Window as... :  saves the current source file to disk.

Close file : closes the current source file.
 

Open explorer output... : loads an explorer output file from disk.

Save explorer output : saves the current explorer output file to disk.

Save explorer output as... :  saves the current explorer output file to disk.
 

View working directories : displays the directories of the environment. (Directories)

Print : opens the print dialog.

Exit : quits the program.
