Edit Menu

Back:  File Menu      Next:   Search Menu       Up:  File Window Contents

Cut (Ctrl-X) : cuts the current selection to the clip board.

Copy (Ctrl-C) : copies the current selection to the clip board.

Paste (Ctrl-V) : pastes the contents of the clip board to the current position.
 

Clear window (Ctrl-Del) : clears the contents of the current window.

Set font : displays a font dialog, where the user can choose the font of the selected text.
