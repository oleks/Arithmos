Set Compiler : selects Borland or GCC as compiler.

Parameters... : shows the Parameters Dialog (Parameter Dialog).

Output format... : shows a dialog to select the output format (Output Format).

Compile... : compiles a C/C++ source file.

Command line arguments... : specifies the arguments for executing the compiled program. (Arguments).

Run... : executes the compiled program.
