Options Menu

Back:  Program Menu      Next:   Help Menu       Up:   File Window Contents

Log on : enables/disables the log.

Clear log file : clears the log file.

Change log file... : selects the filename of the log.

Stack on : enables/disables the stack.

Clear stack : clears the stack.

Load stack... : loads a stack from disk.

Save stack : saves the current stack to disk.

Save stack as... : saves the current stack to disk.

Save options on exit : enables/disables the saving of options on program exit.

Reset to default options : loads the default options.
